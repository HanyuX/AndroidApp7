When you type in the activity type and the data in the start tab using the manual entry, you can see it in the history tab. You can choose to delete the items you do not want. You can also use setting tab to change the unit preference, which can be displayed in the map.
When you type in the GPS, it will show the map and can record your current location. When you save the map, the data will be stored and you can see the item in the history tab.
When you select the automatic mode, the textview will show your current activity.
When you click the sync button, you will upload your history to the server.
